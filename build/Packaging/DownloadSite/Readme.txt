Documentation
=============

Agent install documentation can be found at https://docs.newrelic.com/docs/agents/net-agent/installation/new-relic-net-agent-install-introduction.

File Descriptions
=================

.NET Framework Agent (Windows)
------------------------------
newrelic-agent-win-x64-VERSIONNUMBER.msi
    MSI for installing the 64 bit version of the .NET Framework agent on Windows.

newrelic-agent-win-x86-VERSIONNUMBER.msi
    MSI for installing the 32 bit version of the .NET Framework agent on Windows.

newrelic-agent-win-x64-VERSIONNUMBER.zip
    Zip archive for installing the 64 bit version of the .NET Framework agent on Windows.

newrelic-agent-win-x86-VERSIONNUMBER.zip
    Zip archive for installing the 32 bit version of the .NET Framework agent on Windows.

newrelic-agent-win-VERSIONNUMBER-scriptable-installer.zip
    Zip archive for installing the .NET Framework agent via a scriptable installer.

newrelic-netcore20-agent-win-x64-VERSIONNUMBER.zip
    Zip archive for installing the 64 bit version of the .NET Core agent on Windows.

newrelic-netcore20-agent-win-x86-VERSIONNUMBER.zip
    Zip archive for installing the 32 bit version of the .NET Core agent on Windows.

newrelic-netcore20-agent-win-VERSIONNUMBER-scriptable-installer.zip

    DEB package for installing the .NET Core agent on supported Linux platforms.

newrelic-netcore20-agent-VERSIONNUMBER_amd64.tar.gz
    Tar archive for installing the .NET Core agent on supported Linux platforms.
