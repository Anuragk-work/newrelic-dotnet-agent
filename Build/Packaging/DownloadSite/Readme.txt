Documentation
=============
Windows specific installation information can be found at https://docs.newrelic.com/docs/agents/net-agent/installation/install-net-agent-windows.
Linux specific installation information can be found at https://docs.newrelic.com/docs/agents/net-agent/installation/install-net-core-20-agent-linux.

General agent documentation can be found at https://docs.newrelic.com/docs/agents/net-agent.

File Descriptions:
==================

Full .NET Framework Agent (Windows)
-----------------------------------
newrelic-agent-win-x64-VERSIONNUMBER.msi
    A .msi file to install the 64 bit version of the framework agent on Windows.

newrelic-agent-win-x86-VERSIONNUMBER.msi
    A .msi file to install the 32 bit version of the framework agent on Windows.

newrelic-agent-win-x64-VERSIONNUMBER.zip
    A .zip file containing the nesessary files to manually install the 64 bit version of the framework agent on Windows.

newrelic-agent-win-x86-VERSIONNUMBER.zip
    A .zip file containing the nesessary files to manually install both the 32 bit version of the framework agent on Windows.

newrelic-agent-win-VERSIONNUMBER-scriptable-installer.zip
    A .zip file containing the nesessary files to  the necesary files to install the framework agent via a scriptable installer.


.NET Core Framework Agent (Windows, Linux)
------------------------------------------
newrelic-netcore20-agent-win-x64-VERSIONNUMBER.zip
    A .zip file containing the nesessary files to  manually install the 64 bit version of the .NET Core agent on Windows.

newrelic-netcore20-agent-win-x86-VERSIONNUMBER.zip
    A .zip file containing the nesessary files to  manually install the 32 bit version of the .NET Core agent on Windows.

newrelic-netcore20-agent-win-VERSIONNUMBER-scriptable-installer.zip
    A .zip file containing the nesessary files to install the .NET Core agent via a scriptable installer.

newrelic-netcore20-agent-VERSIONNUMBER.x86_64.rpm
    RPM package containing the necssary files to install the .NET Core agent on a supported Linux platform.

newrelic-netcore20-agent-VERSIONNUMBER_amd64.deb
    DEB package containing the necessary files to install the .NET Core agent on a supported Linux platform.

newrelic-netcore20-agent-VERSIONNUMBER_amd64.tar.gz
    A file containing the nessary files to manually install the .NET Core agent on a supported Linux platform.
